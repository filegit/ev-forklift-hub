## 🧪 完整测试流程

### 测试场景 1：发帖 → 查看列表 → 查看详情 → 点赞

```bash
# 1. 登录获取 Token
TOKEN=$(curl -s -X POST http://localhost:8081/api/login \
  -H "Content-Type: application/json" \
  -d '{"username":"testuser001","password":"123456"}' \
  | jq -r '.data.token')

echo "Token: $TOKEN"

# 2. 发布帖子
curl -X POST http://localhost:8082/api/post \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $TOKEN" \
  -d '{
    "title": "测试帖子标题",
    "content": "这是测试帖子的内容",
    "category": 1
  }'

# 3. 查看帖子列表
curl http://localhost:8082/api/post/list

# 4. 查看帖子详情（假设 ID 为 1）
curl http://localhost:8082/api/post/1

# 5. 点赞帖子
curl -X POST http://localhost:8082/api/post/1/like \
  -H "Authorization: Bearer $TOKEN"

# 6. 再次查看详情，验证点赞数
curl http://localhost:8082/api/post/1
```

### 测试场景 2：多次发帖测试分页

## 🐛 常见问题

### 1. 服务启动失败

**检查：**
- Docker 服务是否运行
- MySQL 是否启动
- Redis 是否启动
- Nacos 是否启动

### 2. 发帖返回 401

**原因：** Token 无效或过期

**解决：** 重新登录获取新 Token

### 3. 发帖返回 400

**原因：** 参数验证失败

**检查：**
- title 不能为空
- content 不能为空
- category 不能为空且必须是数字

### 4. 查询帖子返回空列表

**原因：** 数据库中没有数据

**解决：** 先发布几个帖子
