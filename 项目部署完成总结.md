### 第四步：开发 API 接口

#### 推荐开发顺序：

1. **用户服务 API**
   ```
   POST   /user/register      # 用户注册
   POST   /user/login         # 用户登录
   GET    /user/info          # 获取用户信息
   PUT    /user/info          # 更新用户信息
   ```

2. **社区服务 API**
   ```
   POST   /community/post              # 发布帖子
   GET    /community/post/list         # 帖子列表
   GET    /community/post/{id}         # 帖子详情
   POST   /community/post/{id}/comment # 发表评论
   POST   /community/post/{id}/like    # 点赞
   GET    /community/search            # 搜索帖子
   ```

3. **配件服务 API**
   ```
   GET    /parts/list         # 配件列表
   GET    /parts/{id}         # 配件详情
   POST   /parts/order        # 创建订单
   ```

4. **维修服务 API**
   ```
   POST   /service/order      # 创建维修工单
   GET    /service/order/list # 工单列表
   GET    /service/order/{id} # 工单详情
   PUT    /service/order/{id} # 更新工单状态
   ```
