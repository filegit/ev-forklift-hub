## 🚀 部署到服务器的完整流程

### 第一步：准备服务器

```bash
# 安装 Docker
curl -fsSL https://get.docker.com | sh

# 安装 Docker Compose
sudo curl -L "https://github.com/docker/compose/releases/latest/download/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
sudo chmod +x /usr/local/bin/docker-compose
```

### 第二步：上传项目

```bash
# 方式一：Git
git clone https://github.com/your-repo/ev-forklift-hub.git
cd ev-forklift-hub

# 方式二：SCP
scp -r ev-forklift-hub user@server:/opt/
```

### 第三步：修改配置

编辑 `docker-compose.yml`，修改密码：
- MySQL 密码
- Redis 密码
- Nacos 密码

### 第四步：一键部署

```bash
# Linux
chmod +x deploy.sh
./deploy.sh

# Windows
deploy.bat
```

### 第五步：访问应用

```
前端：http://your-server-ip
网关：http://your-server-ip:8080
Nacos：http://your-server-ip:8848/nacos
```

```bash
# 1. 创建新分支
git checkout -b feature/new-feature

# 2. 开发新功能

# 5. 合并到主分支

```

### 2. 更新到服务器

```bash
# SSH 到服务器
ssh user@your-server-ip

# 进入项目目录
cd /opt/ev-forklift-hub

# 方式一：更新所有服务
./update.sh

# 方式二：只更新特定服务
./update.sh web              # 只更新前端
./update.sh gateway          # 只更新网关
./update.sh user-service     # 只更新用户服务
./update.sh community-service # 只更新社区服务
```

### 3. 验证更新

```bash

# 查看日志

# 测试接口
```
