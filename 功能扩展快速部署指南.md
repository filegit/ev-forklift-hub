# 🚀 功能扩展快速部署指南

## 📋 部署前检查

- ✅ 后端服务已启动（UserApplication、CommunityApplication、GatewayApplication）
- ✅ MySQL、Redis、Nacos 已启动
- ✅ 前端开发服务已启动或已构建

---

## 🔧 部署步骤

### 第一步：创建数据库表

```bash
# 方式一：使用 Docker 执行 SQL 脚本
docker exec efh-mysql mysql -uroot -p123456 < docker/mysql/init/03-create-new-tables.sql

# 方式二：手动在 MySQL 中执行
# 1. 进入 MySQL 容器
docker exec -it efh-mysql mysql -uroot -p123456

# 2. 复制 03-create-new-tables.sql 中的 SQL 语句执行
```

### 第二步：重启后端服务

**在 IDEA 中：**
1. 停止 UserApplication
2. 停止 CommunityApplication
3. 停止 GatewayApplication
4. 重新启动这三个应用

**或使用 Docker：**
```bash
docker-compose restart gateway user-service community-service
```

### 第三步：重新构建前端

```bash
# 进入前端目录
cd efh-web

# 重新构建
npm run build

# 重新构建 Docker 镜像
docker build -t efh-web:latest .

# 重启前端容器
docker-compose restart web
```

### 第四步：验证部署

```bash
# 1. 检查后端服务是否启动
curl http://localhost:8080/user/api/test

# 2. 访问前端
# http://localhost:3000 (开发模式)
# http://localhost (Docker 部署)

# 3. 测试新功能
# - 登录用户
# - 发布帖子
# - 点赞帖子
# - 收藏帖子
# - 访问个人中心
```

## 🐛 常见问题排查

### 问题1：新表未创建

**症状：** 访问点赞、收藏功能时报错 "table not found"

**解决：**
```bash
# 检查表是否存在
docker exec efh-mysql mysql -uroot -p123456 -e "SHOW TABLES FROM efh_community_0 LIKE 'post_like';"

# 如果不存在，手动创建
docker exec efh-mysql mysql -uroot -p123456 < docker/mysql/init/03-create-new-tables.sql
```

### 问题2：前端无法调用后端 API

**症状：** 点赞、收藏等功能不工作

**解决：**
```bash
# 1. 检查网关是否启动
curl http://localhost:8080/user/api/test

# 2. 检查前端 API 路径是否正确
# 应该是 /api/like/post/{postId}，不是 /community/api/like/post/{postId}

# 3. 检查 Token 是否正确传递
# 在浏览器开发者工具中查看请求头
```

### 问题3：个人中心页面无法访问

**症状：** 点击"个人中心"后页面空白或报错

**解决：**
```bash
# 1. 检查路由是否正确配置
# 在 src/router/index.js 中验证 Profile 路由

# 2. 检查浏览器控制台是否有错误
# F12 打开开发者工具，查看 Console 标签

# 3. 检查后端 API 是否返回数据
curl -H "Authorization: Bearer {token}" http://localhost:8080/profile
```

### 问题4：积分显示为 0

**症状：** 用户积分始终为 0

**解决：**
```bash
# 1. 检查 user_points 表是否有数据
docker exec efh-mysql mysql -uroot -p123456 -e "SELECT * FROM user_points WHERE user_id = 1;"

# 2. 如果没有数据，需要在发布帖子时自动创建
# 或手动插入初始数据
docker exec efh-mysql mysql -uroot -p123456 -e "INSERT INTO user_points (user_id, total_points, available_points) VALUES (1, 100, 100);"
```

