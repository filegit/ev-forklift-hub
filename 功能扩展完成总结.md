## 🔧 API 接口清单

### 点赞接口
```
POST   /api/like/post/{postId}           - 帖子点赞/取消
GET    /api/like/post/{postId}/check     - 检查帖子是否已点赞
POST   /api/like/comment/{commentId}     - 评论点赞/取消
GET    /api/like/comment/{commentId}/check - 检查评论是否已点赞
```

### 收藏接口
```
POST   /api/collection/{postId}          - 收藏/取消收藏
GET    /api/collection/check/{postId}    - 检查是否已收藏
GET    /api/collection/my                - 获取我的收藏
```

### 积分接口
```
GET    /api/points                       - 获取用户积分
POST   /api/points/exchange/{exchangeId} - 兑换积分
GET    /api/points/exchanges             - 获取兑换记录
```

### 个人中心接口
```
GET    /api/profile                      - 获取个人信息
PUT    /api/profile                      - 更新个人信息
GET    /api/profile/posts                - 获取我的帖子
GET    /api/profile/comments             - 获取我的评论
```

## 🚀 部署步骤

### 1. 数据库初始化
```bash
# 新表会自动通过 03-create-new-tables.sql 创建
docker exec efh-mysql mysql -uroot -p123456 < docker/mysql/init/03-create-new-tables.sql
```

### 2. 重启后端服务
```bash
# 重启所有后端服务
docker-compose restart gateway user-service community-service

# 或在 IDEA 中重启应用
```

### 3. 重新构建前端
```bash
cd efh-web
npm run build
docker build -t efh-web:latest .
docker-compose restart web
```

## 📝 功能使用说明

### 点赞功能
1. 在帖子详情页点击"点赞"按钮
2. 按钮变为"已赞"状态
3. 再次点击取消点赞
4. 点赞数实时更新

### 收藏功能
1. 在帖子详情页点击"收藏"按钮
2. 按钮变为"已收藏"状态
3. 在个人中心"我的收藏"查看所有收藏
4. 支持取消收藏

### 评论点赞
1. 在帖子详情页评论区点击评论下的"👍"按钮
2. 评论点赞数实时更新
3. 支持取消点赞

### 个人中心
1. 点击右上角用户菜单 → "个人中心"
2. 查看个人信息、统计数据
3. 编辑个人信息
4. 查看我的帖子、评论、收藏
5. 查看积分和兑换记录

### 积分系统
1. 发布帖子获得积分（需后端配置）
2. 发表评论获得积分（需后端配置）
3. 点赞获得积分（需后端配置）
4. 在个人中心"积分兑换"使用积分兑换商品
5. 查看兑换记录

## 📊 数据库表结构

### post_like（帖子点赞表）
- id: 主键
- post_id: 帖子ID
- user_id: 用户ID
- status: 状态（1-已赞，0-已取消）
- create_time: 创建时间
- update_time: 更新时间

### post_collection（帖子收藏表）
- id: 主键
- post_id: 帖子ID
- user_id: 用户ID
- status: 状态（1-已收藏，0-已取消）
- create_time: 创建时间
- update_time: 更新时间

### comment_like（评论点赞表）
- id: 主键
- comment_id: 评论ID
- user_id: 用户ID
- status: 状态（1-已赞，0-已取消）
- create_time: 创建时间
- update_time: 更新时间

### user_points（用户积分表）
- id: 主键
- user_id: 用户ID
- total_points: 总积分
- used_points: 已使用积分
- available_points: 可用积分
- create_time: 创建时间
- update_time: 更新时间

### points_exchange（积分兑换表）
- id: 主键
- user_id: 用户ID
- item_name: 兑换物品名称
- points: 消耗积分
- status: 状态（pending-待发货，completed-已完成，cancelled-已取消）
- create_time: 创建时间
- update_time: 更新时间
