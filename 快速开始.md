# 新能源叉车社区平台 - 快速开始

## 🎯 5分钟快速启动

### 前置条件
确保以下软件已安装并启动：
- ✅ JDK 1.8+
- ✅ Maven 3.6+
- ✅ MySQL 8.0+ (运行中)
- ✅ Redis 6.0+ (运行中)
- ✅ Nacos 2.x (运行中)

### 第一步：初始化数据库

```sql
-- 1. 创建数据库
CREATE DATABASE efh_user_0 DEFAULT CHARACTER SET utf8mb4;
CREATE DATABASE efh_user_1 DEFAULT CHARACTER SET utf8mb4;
CREATE DATABASE efh_community_0 DEFAULT CHARACTER SET utf8mb4;
CREATE DATABASE efh_community_1 DEFAULT CHARACTER SET utf8mb4;
CREATE DATABASE efh_parts DEFAULT CHARACTER SET utf8mb4;
CREATE DATABASE efh_service DEFAULT CHARACTER SET utf8mb4;

-- 2. 执行 database.sql 脚本
source database.sql;
```

### 第二步：修改配置

修改各服务 `application.yml` 中的数据库密码：

```yaml
spring:
  shardingsphere:
    datasource:
      ds0:
        username: root
        password: 你的密码  # 修改这里
```

### 第三步：启动项目

**方式一：使用脚本（推荐）**
```bash
双击运行: 快速启动脚本.bat
```

**方式二：IDEA 启动**
1. 运行 GatewayApplication (8080)
2. 运行 UserApplication (8081)
3. 运行 CommunityApplication (8082)
4. 运行 PartsApplication (8083)
5. 运行 ServiceApplication (8084)

### 第四步：验证

访问 Nacos 控制台：http://localhost:8848/nacos
- 账号：nacos
- 密码：nacos

检查所有服务是否注册成功。

## 🔧 技术栈

- **微服务框架**: Spring Cloud Alibaba
- **服务注册**: Nacos
- **API 网关**: Spring Cloud Gateway
- **数据库**: MySQL 8.0 + ShardingSphere (分库分表)
- **缓存**: Redis + Redisson (分布式锁)
- **消息队列**: Kafka
- **搜索引擎**: Elasticsearch
- **ORM 框架**: MyBatis-Plus

## ⚠️ 注意事项

1. **端口占用**: 确保 8080-8084 端口未被占用
2. **数据库**: 必须先创建数据库并执行建表脚本
3. **Nacos**: 必须先启动 Nacos 服务
4. **启动顺序**: 建议先启动网关，再启动其他服务
5. **内存**: 建议至少 4GB 可用内存

## 🐛 常见问题

### 1. 服务启动失败
- 检查 Nacos 是否启动
- 检查数据库连接信息
- 查看控制台错误日志

### 2. 无法访问接口
- 检查服务是否在 Nacos 中注册成功
- 检查网关路由配置
- 检查防火墙设置

### 3. 数据库连接失败
- 确认数据库已创建
- 确认用户名密码正确
- 确认 MySQL 服务已启动
