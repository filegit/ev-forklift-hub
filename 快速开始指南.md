# 🚀 EV-Forklift-Hub 快速开始指南

## 📋 5分钟快速启动

### 前提条件

- ✅ Docker 已安装并运行
- ✅ IDEA 已安装
- ✅ JDK 8 已安装
- ✅ Maven 已配置

---

## 🎯 第一步：启动 Docker 基础设施（2分钟）

```bash
# 1. 进入项目目录
cd d:\LH\ev-forklift-hub

# 2. 启动 MySQL、Redis、Nacos
docker-compose up -d mysql redis nacos

# 3. 等待服务启动（约 1-2 分钟）
# 查看状态
docker-compose ps
```

**预期结果：**
```
NAME        STATUS      PORTS
efh-mysql   Up          0.0.0.0:3306->3306/tcp
efh-redis   Up          0.0.0.0:6379->6379/tcp
efh-nacos   Up          0.0.0.0:8848->8848/tcp
```

---

## 🎯 第二步：修复数据库表结构（1分钟）

```bash
# 连接到 MySQL
docker exec -it efh-mysql mysql -uroot -p123456

# 执行以下 SQL（复制粘贴）：
USE efh_user_0;
ALTER TABLE `user` ADD COLUMN `user_type` TINYINT DEFAULT 1 AFTER `gender`;
ALTER TABLE `user` ADD COLUMN `points` INT DEFAULT 0 AFTER `status`;
ALTER TABLE `user` MODIFY COLUMN `id` BIGINT NOT NULL AUTO_INCREMENT;

USE efh_user_1;
ALTER TABLE `user` ADD COLUMN `user_type` TINYINT DEFAULT 1 AFTER `gender`;
ALTER TABLE `user` ADD COLUMN `points` INT DEFAULT 0 AFTER `status`;
ALTER TABLE `user` MODIFY COLUMN `id` BIGINT NOT NULL AUTO_INCREMENT;

exit;
```

---

## 🎯 第三步：启动微服务（2分钟）

### 在 IDEA 中依次启动：

1. **UserApplication** (efh-user)
   - 等待看到：`========== 用户服务启动成功 ==========`

2. **CommunityApplication** (efh-community)
   - 等待看到：`========== 社区服务启动成功 ==========`

3. **GatewayApplication** (efh-gateway)
   - 等待看到：`========== Gateway服务启动成功 ==========`

---

## 🎯 第四步：验证服务（30秒）

### 1. 验证 Nacos
访问：http://localhost:8848/nacos  
登录：nacos / nacos  
切换到 dev 命名空间，应该看到 3 个服务

### 2. 测试用户服务
```bash
curl http://localhost:8081/api/test
```

### 3. 测试社区服务
```bash
curl http://localhost:8082/api/post/test
```

### 4. 测试网关
```bash
curl http://localhost:8080/user/api/test
```

---

## 🎯 第五步：测试完整流程（2分钟）

### 1. 注册用户
```bash
curl -X POST http://localhost:8081/api/register \
  -H "Content-Type: application/json" \
  -d '{"username":"testuser","password":"123456","nickname":"测试用户","phone":"13800138000"}'
```

**保存返回的 Token！**

### 2. 登录
```bash
curl -X POST http://localhost:8081/api/login \
  -H "Content-Type: application/json" \
  -d '{"username":"testuser","password":"123456"}'
```

### 3. 发布帖子
```bash
curl -X POST http://localhost:8082/api/post \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN_HERE" \
  -d '{"title":"测试帖子","content":"这是测试内容","category":1}'
```

### 4. 查看帖子列表
```bash
curl http://localhost:8082/api/post/list
```

### 5. 查看帖子详情
```bash
curl http://localhost:8082/api/post/1
```

## 🐛 常见问题

### Q: Docker 服务启动失败？
**A:** 检查 Docker Desktop 是否运行，端口是否被占用

### Q: Nacos 无法访问？
**A:** 等待 1-2 分钟，Nacos 启动较慢

### Q: 微服务启动失败？
**A:** 检查 MySQL 和 Redis 是否启动成功

### Q: 接口返回 401？
**A:** Token 无效或过期，重新登录获取新 Token

### Q: 数据库连接失败？
**A:** 确保执行了第二步的数据库修复脚本
