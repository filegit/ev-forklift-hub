# 新能源叉车售后论坛及零部件交易平台 - 启动指南

## 一、环境准备

### 1. 必需软件
- **JDK 8** ✓ (已安装)
- **Maven 3.6+** ✓ (已安装)
- **MySQL 8.0+** (需要安装)
- **Redis 6.0+** (可选，如不使用需修改配置)

### 2. 数据库初始化

#### 启动MySQL服务
确保MySQL服务正在运行。

#### 执行数据库脚本
```bash
mysql -u root -p < database.sql
```

这将创建：
- 数据库：`ev_forklift_hub`
- 6个数据表：user, post, comment, parts, parts_order, service_order
- 3个测试账号：admin, technician, merchant

### 3. 配置文件修改

编辑 `efh-user/src/main/resources/application.yml`：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/ev_forklift_hub?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai
    username: root
    password: 你的MySQL密码  # 修改这里
  redis:
    host: localhost
    port: 6379
```

**如果没有Redis**，可以暂时注释掉Redis配置，项目仍可运行（但缓存功能不可用）。

## 二、启动应用

### 方式1：使用Maven启动（推荐开发环境）

```bash
cd efh-user
mvn spring-boot:run
```

### 方式2：打包后启动（推荐生产环境）

```bash
# 在项目根目录执行
mvn clean package

# 启动应用
java -jar efh-user/target/efh-user-1.0.0.jar
```

### 启动成功标志

看到以下日志表示启动成功：
```
Started UserApplication in X.XXX seconds
```

默认端口：**8080**

访问地址：http://localhost:8080

## 三、API接口测试

### 1. 用户注册
```bash
curl -X POST http://localhost:8080/user/register ^
  -H "Content-Type: application/json" ^
  -d "{\"username\":\"testuser\",\"password\":\"123456\",\"nickname\":\"测试用户\",\"phone\":\"13800138000\"}"
```

### 2. 用户登录
```bash
curl -X POST http://localhost:8080/user/login ^
  -H "Content-Type: application/json" ^
  -d "{\"username\":\"admin\",\"password\":\"admin\"}"
```

返回示例：
```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiJ9..."
  }
}
```

**重要**：复制返回的token，后续需要认证的接口都需要使用。

### 3. 获取用户信息（需要Token）
```bash
curl -X GET http://localhost:8080/user/info ^
  -H "Authorization: 你的token"
```

### 4. 发布帖子（需要Token）
```bash
curl -X POST http://localhost:8080/community/post ^
  -H "Content-Type: application/json" ^
  -H "Authorization: 你的token" ^
  -d "{\"title\":\"叉车电池保养技巧\",\"content\":\"分享一些电池保养的经验...\",\"category\":3}"
```

### 5. 查看帖子列表（无需Token）
```bash
curl -X GET "http://localhost:8080/community/post/list?page=1&size=10"
```

### 6. 发布零部件（需要Token）
```bash
curl -X POST http://localhost:8080/parts ^
  -H "Content-Type: application/json" ^
  -H "Authorization: 你的token" ^
  -d "{\"name\":\"电池组\",\"description\":\"48V锂电池\",\"category\":\"电池\",\"brand\":\"比亚迪\",\"model\":\"BYD-48V\",\"price\":5000,\"stock\":10}"
```

### 7. 查看零部件列表（无需Token）
```bash
curl -X GET "http://localhost:8080/parts/list?page=1&size=10"
```

### 8. 创建服务工单（需要Token）
```bash
curl -X POST http://localhost:8080/service/order ^
  -H "Content-Type: application/json" ^
  -H "Authorization: 你的token" ^
  -d "{\"serviceType\":1,\"title\":\"叉车无法启动\",\"description\":\"按下启动按钮没有反应\",\"address\":\"北京市朝阳区XX路XX号\",\"phone\":\"13800138000\"}"
```

## 五、常见问题

### 1. 端口被占用
如果8080端口被占用，修改 `application.yml` 中的端口：
```yaml
server:
  port: 8081  # 改为其他端口
```

### 2. 数据库连接失败
- 检查MySQL服务是否启动
- 检查用户名密码是否正确
- 检查数据库 `ev_forklift_hub` 是否已创建

### 3. Redis连接失败
如果没有安装Redis，可以暂时注释掉 `efh-common/pom.xml` 中的Redis依赖：
```xml
<!-- 暂时注释
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-data-redis</artifactId>
</dependency>
-->
```

### 4. 编译错误
重新编译：
```bash
mvn clean compile
```

## 六、项目结构说明

```
ev-forklift-hub/
├── efh-common/          # 公共模块（工具类、配置、异常处理）
├── efh-user/            # 用户模块（注册、登录、认证）★ 主启动模块
├── efh-community/       # 社区论坛模块
├── efh-parts/           # 零部件交易模块
├── efh-service/         # 售后服务模块
├── database.sql         # 数据库初始化脚本
└── README.md            # 项目说明文档
```

**注意**：只需要启动 `efh-user` 模块，它会自动加载所有其他模块的功能。
